## Cards requirements

### Card creation
[x] User is able to store a card
[x] User is not able to store an invalid card
[x] User is not able to store a card if not authenticated
[x] A user card list, if empty, is able to set its first card as main
[x] A user card list is able to have only one of its cards as main

### Card index
[x] Admin User is able to list all stored cards, regardless of who created them
[x] Admin User is able to see the first page of stored cards
[x] Admin User is able to see the second page of stored cards
[x] User is able to list all his stored cards

### Card show
[ ] User is able to check a card details
[ ] User is not able to check an invalid card details
[ ] User is not able to check a card he does not own

### Card deletion
[ ] User is able to delete his own card
[ ] User is not able to delete a card he does not own

### Card expiration
[ ] Notify user when his card is about to expire (15 days before expiration)
[ ] Mark card as invalid when after its expiration date

### Card main
[ ] User is be able to update his cards to main
[ ] User card list is able to have only one of its cards as main
